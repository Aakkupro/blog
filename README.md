# Blog App

## Description

The Blog App is a web application that allows users to:

- Create, view, edit, and delete blog posts.
- Like, comment on, and share posts.

The app features a responsive design, ensuring a seamless experience on various devices.

## Features

- **User Authentication:**
    - Users can sign up and log in to the app.
    - Only logged-in users can create, edit, and delete posts.
- **Blog Post Management:**
    - Users can create new blog posts with titles and content.
    - Posts are displayed on the homepage and user profile pages.
    - Users can edit and delete their own posts.
- **Liking System:**
    - Users can like posts to show appreciation.
    - The total number of likes is displayed for each post.
- **Commenting System:**
    - Users can add comments to posts.
    - Comments are displayed below each post.
- **Sharing:**
    - Users can share posts on social media.
- **Responsive Design:**
    - The app adapts to different screen sizes, providing an optimal viewing experience on desktops, tablets, and mobile devices.
- **User Profile**
    - Each user have their profile to see their post and likes and many more.
- **Save Blog**
    - User can save their blog to read them later.
- **Search**
    - User can search blogs.
- **Categories**
    - Blog are categoriesed in different categories.

## Future Enhancements

- **Real-time Updates:** Implement real-time updates for new posts, likes, and comments using technologies like WebSockets.
- **Notifications:** Notify users of new comments, likes, and follows.
- **Image and Video Support:** Allow users to include images and videos in their blog posts.
- **Search Engine Optimization (SEO):** Improve the app's visibility on search engines.
- **Admin Dashboard:** Create an admin dashboard to manage users, posts, and comments.

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage

## Installation

1. Clone the repository.
2. Open the `index.html` file in your web browser.

## Usage

1. **Sign Up / Log In:**
   - If you're a new user, click on "Sign Up" to create an account.
   - If you're an existing user, enter your email and password to log in.
2. **Create a Blog Post:**
   - Click on the "Create Blog" button.
   - Enter a title and content for your post.
   - Click on "Post" to publish your blog.
3. **View Blog Posts:**
   - The homepage displays all the latest blog posts.
   - Click on a post to view it in detail.
4. **Like, Comment, and Share:**
   - Use the buttons below each post to like, comment, or share.
5. **Edit or Delete Your Posts:**
   - Go to your profile page.
   - Click on the "Edit" or "Delete" button next to the post you want to modify.
